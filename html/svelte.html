Svelte

TSvelte is an innovative JavaScript compiler designed to produce high-performance user interfaces. Rich Harris created it in 2017 and it is still in its early stages of development. It is distinct in that it does not employ a virtual DOM. Instead, it employs a specialised JavaScript Virtual Machine designed specifically for creating user interfaces.
